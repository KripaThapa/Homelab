# Project Overview

## What we are building
We are building a shared Raspberry Pi home lab platform using 4 Raspberry Pi 5 devices and 1 TB of external storage.

The long-term goal is to host a Family Dashboard / Life Management System on top of this platform.

## Why we are building it
We want to learn system design by building and operating a real system that we will actually use.

## Learning goals
- Linux and Raspberry Pi administration
- Networking and node management
- Kubernetes / cluster setup
- Ingress and routing
- Monitoring and observability
- GitOps deployment
- Databases and persistence
- Backend and frontend application deployment

## Constraints
- Weekend-only project
- Approximately 12 team-hours per weekend
- Keep scope small enough to finish

## Phase 1 Goal
Prepare all Raspberry Pis, networking, storage, and documentation so the cluster can be built in Week 2.
